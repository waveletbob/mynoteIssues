安装使用
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

    nvm ls
    nvm install v14
    nvm use v14
    node -v
    npm -v
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    cnpm install vue -g
    cnpm install vue-cli -g
    cnpm install webpack -g
    vue init webpack admin-front
    npm run dev
