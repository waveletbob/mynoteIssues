# 自进化 Agent

## 核心思路

让 Agent 从执行结果中学习，逐步提升决策质量。本质是**反馈闭环 + 经验积累**。

## 关键机制

```
执行任务 → 记录结果 → 自我评估 → 提取经验 → 更新策略 → 下次更好
```

### 实现路径

| 层级 | 做法 | 工具/方法 |
|------|------|-----------|
| **反思** | 每轮执行后自我评价 | CoT 自我批判 |
| **记忆** | 经验存入向量库 | RAG + 向量数据库 |
| **策略更新** | 用经验微调 prompt | 动态 prompt 模板 |

## 实用模式

```
执行前：检索历史相似任务的经验
执行中：记录关键决策和中间结果
执行后：对比预期和实际，总结教训
下次用：注入教训到 system prompt
```

## 注意

- 自进化目前更多是**工程优化**（经验注入、prompt 调优），而非真正的模型自我改进
- 核心价值在**降低重复错误率**，不是创造新能力
- 落地建议：先从简单的反思机制开始，别一步到"完全自治"
