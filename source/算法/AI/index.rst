技术栈
--------

**开发框架**:
- PyTorch、TensorFlow、JAX
- Hugging Face Transformers
- LangChain、LlamaIndex

**训练工具**:
- DeepSpeed、Megatron-LM
- vLLM、FlashAttention

**部署平台**:
- Ollama、LocalAI
- vLLM、TensorRT-LLM

**协议和框架**:
- MCP (Model Context Protocol)
- SKILL (技能系统)
- Agent框架

**评估工具**:
- GLUE、SuperGLUE
- MMLU、HumanEval

学习路径
----------

1. **基础入门**: 从 :doc:`ai-overview` 开始了解AI技术全景
2. **深入LLM**: 学习 :doc:`llm-overview` 掌握大语言模型
3. **实践应用**: 通过 :doc:`prompt-engineering` 学习提示工程
4. **进阶开发**: 掌握 :doc:`ai-agent` 和 :doc:`rag-technology`
5. **协议集成**: 学习 :doc:`mcp-technology` 和 :doc:`skill-system`

核心技术
---------

**大语言模型 (LLM)**:
- Transformer架构
- 预训练和微调
- 推理优化
- 模型评估

**检索增强生成 (RAG)**:
- 向量数据库
- 文档处理
- 检索优化
- 知识图谱

**AI Agent**:
- 智能体架构
- 工具使用
- 推理能力
- 多Agent协作

**多模态AI**:
- 视觉语言模型
- 文生图技术
- 音频处理
- 跨模态理解

**MCP协议**:
- 协议架构
- 工具开发
- 资源管理
- 安全机制

**SKILL系统**:
- 技能定义
- 技能组合
- 技能学习
- 技能评估
